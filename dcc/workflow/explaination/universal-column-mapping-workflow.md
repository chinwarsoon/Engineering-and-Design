# Universal Column Mapping Workflow

## Overview
Universal column mapping workflow with fuzzy matching and schema integration.

## Functions
- **load_main_schema**: Load the main schema configuration
- **detect_columns**: Detect and identify columns in the data
- **rename_dataframe_columns**: Rename dataframe columns according to schema
- **get_column_bounds**: Get column boundaries and metadata

## Tool File
`tools/universal_column_mapper.py`

## Description
This workflow provides universal column mapping capabilities with fuzzy matching to automatically identify and rename columns based on schema definitions. It handles column detection, boundary identification, and automated renaming processes.

## Usage
Used as part of the UNIVERSAL_PROCESSING workflow for the "Universal column mapping" step.
