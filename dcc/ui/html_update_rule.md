# apply the follow requirements to all HTML files
1. title bar should span the whole width of the screen.
2. use side icon bar to toggle sidebar to show related tools
3. side bar should be toggled by clicking the left side of the icon bar.
4. color theme should be toggled by clicking the theme button in the top right corner.
5. color theme should have light (bright backgorund), dark (dark background), sky (sky light blue background), ocean (ocean blue background), and presentation (light grey background) options. make sure the theme is saved in local storage.
