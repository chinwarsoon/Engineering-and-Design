# Universal Document Processing Workflow

## Overview
Universal document processing workflow with schema-driven calculations and null handling.

## Functions
- **load_schema**: Load schema configuration for processing
- **process_data**: Process data according to schema rules
- **apply_null_handling**: Apply null value handling strategies
- **apply_calculations**: Apply calculated fields and transformations
- **validate_data**: Validate processed data against schema

## Tool File
`tools/universal_document_processor.py`

## Description
This workflow provides comprehensive document processing capabilities including schema-driven data processing, null handling strategies, automated calculations, and data validation. It ensures data quality and consistency according to defined schema rules.

## Usage
Used as part of the UNIVERSAL_PROCESSING workflow for the "Document processing" step.
